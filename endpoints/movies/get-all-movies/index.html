Get all movies

Endpoint

  • HTTP Method: GET
  • URL: api/movies/

Description

Retrieves a paginated list of all movies.

Query Parameters

  • pageNumber (optional): Page number of pagination (default: 1).
  • pageSize (optional): Number of items per page (default: 10).

Response

  • HTTP 200 OK: Returns a paginated list of movies.
